# 🔥 Hidden Legacy

🐾 **Claws** es un ninja aventurero que ha escuchado historias sobre un antiguo tesoro oculto en el centro de un misterioso laberinto. Durante generaciones, muchos exploradores han intentado encontrarlo, pero ninguno ha logrado regresar para contar su historia.

🔥 Decidido a descubrir la verdad, Claws se adentra en el laberinto enfrentándose a trampas, caminos engañosos y peligros ocultos en cada rincón. Para sobrevivir, deberá usar su agilidad, reflejos y habilidades de exploración mientras busca la ruta correcta hacia el centro.

🐾 Sin embargo, el laberinto nunca es igual. Cada partida comienza desde una entrada diferente, cambiando completamente el camino hacia el tesoro.

---

## 🎯 Objetivo

🔥 Ahora depende de ti ayudar a **Claws** a encontrar el **Hidden Legacy**.

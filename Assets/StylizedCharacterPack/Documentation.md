This pack contains 5 characters rigged with some animations to use in your projects.

There is 3 type of animations: **small**, **medium** and **big**
    - Small animation are for the **Bat** and **Sloth**
    - Medium animation are for the **Rabbit** and **Leopard**
    - Big animation are for ther **Seagull**

You can find the character as Prefab in the Prefabs/Character folder.
You can find the Animation file (and sample Animation Controller) in the Animation Folder

The **Demo** folder contains a couple of scene in which you can move around to look at the animation.
The Control are :

 - WASD or Arrow to move around
 - Shift to Sprint
 - Space to Jump
 - Mouse move to rotate the camera
